KIMSUFI AVAILABILITY MONITOR

An application to monitor the availability of Kimsufi servers.


REQUIREMENTS

This application requires Microsoft .NET Framework 4.6.1 to run.

https://www.microsoft.com/en-us/download/details.aspx?id=49982


DOWNLOADS

The application can be downloaded from the Releases page of this repository.

https://github.com/kappa7194/kimsufi-availability-monitor/releases


IN CASE OF PROBLEMS

Open an issue detailing the problem AND attaching the application logs.
The logs can be found in the Logs directory inside the application directory.
Compress the Logs directory and attach it to the issue.

https://github.com/kappa7194/kimsufi-availability-monitor/issues
